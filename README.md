# weather-dashboard

## Purpose
A website that allows user search cities for current weather and five day forecast. The website will display the search history alongside the weather information. 

## Built With
* HTML
* CSS
* Javascript
* moments.js
* jquery
* bootstrap
* Openweather APIs
